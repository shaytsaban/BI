📘 CAPM Components & Assumptions

❓ What is Alpha?

Alpha (α) is the difference between the stock's actual annualized return and the return predicted by CAPM. It represents manager/skill or mispricing. α > 0 means the stock beat its risk-adjusted benchmark.

❓ Expected Return?

Calculated as Rf + β × (Market Return − Rf). It's the compensation CAPM says investors should demand for taking on the stock's systematic risk. Based on historical data, not a guarantee.

❓ Why BUY/SELL/HOLD?

Signals are heuristic: BUY if α > 1% (stock outperformed its risk profile), SELL if α < −1%, HOLD if near zero. These are model-based flags, not financial advice.

❓ Key Assumptions

• Linear risk-return relationship
• SPY proxies the true market portfolio
• 4.25% static risk-free rate
• Historical returns proxy future expectations
• Returns are normally distributed (for σ)
